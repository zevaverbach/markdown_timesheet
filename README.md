# Markdown Timesheet

A command line interface to total up the time entries in a markdown file.

## Usage

```
$ pip install markdown_timesheet
...
$ cat timesheet.md

# 5 Sep 2019

## 9:15-9:16, 9:20-9:25
Write spec, write client email.
  - wrote really basic spec
	- told client I'm starting the project

## 9:25-9:29
Write tests

$ add timesheet.md
10
```
